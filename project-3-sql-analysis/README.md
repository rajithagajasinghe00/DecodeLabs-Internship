# Project 3: SQL Data Analysis

**Goal:** practice answering business questions using SQL queries instead
of only pandas.

**What the notebook covers:**
- Loading a CSV into a lightweight SQL database (SQLite, built into Python)
- `SELECT` — choosing which columns to see
- `WHERE` — filtering rows by a condition
- `ORDER BY` — sorting results
- `GROUP BY` with `COUNT`, `SUM`, `AVG` — summarizing by category
- `HAVING` — filtering *after* grouping

**Run it:** open `project3_sql_analysis.ipynb`, click **Open in Colab**,
run each cell in order, and upload your CSV when prompted.
