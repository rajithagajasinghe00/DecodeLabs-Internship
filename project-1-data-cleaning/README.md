# Project 1: Data Cleaning & Preparation

**Goal:** turn a messy raw dataset into a clean, reliable one.

**What the notebook covers:**
- Finding missing/blank values and deciding how to fill them
- Removing duplicate rows and duplicate IDs
- Fixing date formatting
- Cleaning up inconsistent text (extra spaces, mismatched capitalization)
- Saving the cleaned file so it can be reused in Project 2 and 3

**Run it:** open `project1_data_cleaning.ipynb`, click **Open in Colab**,
run each cell in order, and upload your CSV when prompted.
