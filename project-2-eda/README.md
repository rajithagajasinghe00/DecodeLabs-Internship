# Project 2: Exploratory Data Analysis (EDA)

**Goal:** understand what the data is actually telling you before building
anything more advanced on top of it.

**What the notebook covers:**
- Basic statistics (mean, median, min, max) with `.describe()`
- Comparing mean vs. median to spot skewed data
- Visualizing the distribution with a histogram
- Finding outliers with a boxplot
- Grouping and comparing categories (e.g. revenue by product)
- Checking relationships between numeric columns (correlation)

**Run it:** open `project2_eda.ipynb`, click **Open in Colab**, run each
cell in order, and upload your CSV when prompted (the cleaned CSV from
Project 1 works well here, but any similar CSV works too).
